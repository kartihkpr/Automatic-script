- Windows
- Ubuntu

- VS Code

- Python
- Docker Desktop
- Zoom

- VLC

- Node.js
- AnyDesk

## Windows Usage

Run PowerShell as Administrator:

```powershell
Set-ExecutionPolicy Bypass -Scope Process -Force
.\windows-install.ps1
```

## Ubuntu Usage

```bash
chmod +x ubuntu-install.sh
./ubuntu-install.sh
```
